# Quick Start Guide

Get up and running with the AI Chat Assistant in 5 minutes!

## 1. Install & Run

```bash
# Install dependencies
pnpm install

# Start the app
pnpm dev
```

Visit `http://localhost:3000` in your browser.

## 2. Configure API

1. Go to the **Settings** tab
2. Paste your Ollama API key
3. Select a model (e.g., "Llama 2")
4. Click "Save Settings"

## 3. Start Chatting

1. Go to the **Chat** tab
2. Type a message
3. Press Enter to send
4. Watch the response stream in real-time!

## Need Help?

- **Settings not saving?** Check your browser storage permissions
- **API error?** Verify your API key and endpoint are correct
- **Slow responses?** Lower "Max Tokens" in settings
- **Want better prompts?** Use the **Optimizer** tab
